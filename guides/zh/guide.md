# 项目指引

GizClaw 是面向 GizClaw 设备的开箱即用 Agent Runtime 和 Edge Server。它以 WebRTC 作为设备、节点和客户端之间的连接与传输层，在其上提供 Agent Runtime、Admin/RPC API、设备遥测与状态监控、OTA、数字内容、社交和 Gameplay 等服务。
