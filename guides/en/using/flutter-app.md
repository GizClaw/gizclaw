# Flutter App <Badge type="warning" text="WIP" />

This page explains installation, permissions, connected devices, and common operations of the GizClaw Flutter App.

The App requests the current RuntimeProfile Workflow catalog with `source=runtime`. Returned Workflow IDs are stable aliases that the App maps locally to display names and icons; the Server does not provide Workflow icons or i18n. Workspace creation stores the same runtime source and alias.
